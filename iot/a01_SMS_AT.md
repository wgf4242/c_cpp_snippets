## basic

```
AT+CSQ
AT+CPIN?
AT+CPIN="your_PIN_code"
AT+CREG?
AT+COPS?
```

解释
```
AT+CSQ -- 网络信号  0,0 没信号
AT+CPIN?
+CPIN: SIM PIN
OK -- 需要提供pin码

AT+CPIN="your_PIN_code"


AT+CREG?

+CREG: 0,1 -- 已经注册

AT+COPS?
+COPS: 0,2,"46000",7
OK
    460 是中国的国家代码 (MCC)。
    00 是网络代码 (MNC)，代表 中国移动 (China Mobile)。
```
