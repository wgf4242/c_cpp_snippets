# Integer Procedence

| Operator | Name                    | Precedence |
| -------- | ----------------------- | ---------- |
| ()       | Parentheses             | 1          |
| +,-      | Unary plus, Unary minus | 2          |
| *,/      | Multiply, Divide        | 3          |
| MOD      | Modulus                 | 3          |
| +,-      | Add, Subtract           | 4          |
